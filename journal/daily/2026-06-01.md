What I Learned:

**MeTTa Fundamentals:**
- An Atom is the fundamental unit — everything in MeTTa is an atom (data and code unified)
- 4 atom types: Symbol (named concept), Variable ($x — for pattern matching), Expression ((+ 1 2) — compound), Grounded (Python/Rust-backed values like numbers)
- `=` defines rewrite rules, NOT assignment — `(= (add $x $y) (+ $x $y))` means "rewrite (add $x $y) into (+ $x $y)"
- `!` prefix evaluates and returns; no `!` stores into the Atomspace
- MeTTa is non-deterministic — one expression can return multiple results simultaneously
- Variables only mean something during pattern matching — `$x` has no standalone value
- Data and logic share the same syntax — `(isa cat animal)` is both a fact AND queryable
