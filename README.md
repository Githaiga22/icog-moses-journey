# iCog MOSES Journey

My personal learning repository for the **iCog Labs AI Internship Program**, documenting the journey of contributing to the **Hyperion MOSES Project**.

**Internship Period:** June 1 – July 30, 2026
**Team:** Hyperion MOSES

---

## What Is MOSES?

MOSES (Meta-Optimizing Semantic Evolutionary Search) is an evolutionary program synthesis algorithm. The current project focuses on porting MOSES from OpenCog Classic (C++) into the **Hyperon framework** using the **MeTTa programming language**.
