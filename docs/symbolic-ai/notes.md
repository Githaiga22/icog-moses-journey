# Symbolic AI

## What It Is

Symbolic AI represents knowledge through human-readable symbols, rules, and logic rather than numeric weights or pattern statistics.

## Advantages

- Interpretable and explainable
- Learns from small datasets
- Can apply logical rules explicitly
- Supports structured reasoning

## Limitations

- Brittle under noisy or ambiguous input
- Hard to scale to perceptual tasks (vision, speech)
- Requires manual knowledge engineering

## Use Cases

- Knowledge graphs
- Logical reasoning
- Planning and decision-making
- Program synthesis (e.g., MOSES)
