# OpenCog

## Historical Context

OpenCog Classic was an open-source AGI framework built on symbolic AI. It aimed to mimic cognitive processes through modular components.

## Key Components

- **MOSES** — evolutionary program learning
- **PLN (Probabilistic Logic Networks)** — probabilistic reasoning
- **ECAN (Economic Attention Networks)** — attention allocation
- **Pattern Miner** — frequent pattern discovery
- **RelEx / Link Grammar** — natural language processing

## MOSES Integration

In OpenCog Classic, MOSES was used for:
- Predictive modeling
- Agent action selection
- Behavior learning from examples

The goal of the current project is to port MOSES from C++ (OpenCog Classic) into MeTTa (Hyperon).
