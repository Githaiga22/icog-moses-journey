# Hyperon

## Architecture

Hyperon is the next-generation OpenCog framework built around the Atomspace — a typed metagraph for storing and reasoning over knowledge.

Key components:
- **MeTTa** — the primary programming and knowledge representation language
- **Atomspace** — the central knowledge store (typed, distributed)
- **Grounded atoms** — bridge between MeTTa and external systems (Python, Rust)

## Goals

- Provide a unified framework for AGI research
- Enable symbolic + sub-symbolic integration
- Support distributed, scalable reasoning

## Ecosystem

- SingularityNET is the primary sponsor
- Replaces OpenCog Classic
- MOSES is being ported into this framework
